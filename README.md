# Virtual Interior Designer

An AI-powered interior design web application that transforms room photos into redesigned spaces with real-time budget tracking.

## Features
- Upload a room photo and receive instant AI-generated redesign suggestions
- Interactive Before/After slider for visual comparison
- Conversational AI chat for iterative design changes
- Smart budget tracking with category-wise cost breakdown
- Six design styles: Modern, Scandinavian, Bohemian, Industrial, Traditional, Japandi

## Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** Python, FastAPI
- **AI Model:** Groq API — Llama 3.1 8B Instant
- **Image Generation:** Pollinations AI
- **Deployment:** Hugging Face Spaces
